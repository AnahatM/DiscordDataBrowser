# Images Folder

This folder contains assets for the README.

## Files to add:

- `icon.png` - A 100x100 or 200x200 PNG logo for the project (you can convert icon.svg to PNG)
- `screenshot.png` - A screenshot of the Discord Data Browser in action

## Converting SVG to PNG

You can use an online converter or run this command if you have ImageMagick:

```bash
convert icon.svg -resize 200x200 icon.png
```

Or use an online tool like https://svgtopng.com/
